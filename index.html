Workshop

Todo app

Bouw een Todo app met VueJS

Voor de pro's: maak gebruik van VueX, vuePress, Animations, etc.

Wat moet het doen?

  • Nieuw item toevoegen
  • Item afvinken/aanpassen
  • Afgevinkte items verwijderen
  • Maak gebruik van persistence (bvb. localStorage)

Wat heb je nodig?

  • Declarative rendering*
  • Data Binding*
  • Iterating Data*
  • Conditional Rendering*
  • Event Emission*
  • Vue CLI*

Gettng started

  1. npm init
  2. npm install vue-cli
  3. ./node_modules/.bin/vue init webpack my-todo-app
  4. cd my-todo-app
  5. npm start
  6. Implementeer evt. een UI library (bootstrap/bulma/etc.)
